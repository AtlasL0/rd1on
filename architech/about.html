Functions

name(user_name):
Sets the model's name and returns a loading message.
Example:

hello(user_name, choice):
Greets the user based on a predefined choice.
Example:

  • user_name (str): The name to set for the model.
  • choice (str): The greeting choice (a1, a2, or a3).
    a1: "Hello! How may I help you?"
    a2: "Hello. How may I assist you today?"
    a3: "yo. what's up."

如果你需要展示的内容太多，可以把它隐藏在收缩框里，需要时再把它展开。( display 不能包含英文逗号，可用&sbquo;)
