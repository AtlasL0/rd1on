About

A Python package for natural language processing.

Installation

To use this code, make sure you have the following dependencies installed:

  • Python 3.8 or newer: www.python.org
  • Colorama: pip install colorama

To install Architech, run the following command in your terminal:
pip install architech

To import it to your Python script, simply add the following code:
import architech

Getting Started

Multiple sections are included in the Architech package.
If you are new to Architech, this part of the documentation is meant for you.

from architech import start

Functions

  • name(user_name):
    Sets the model's name and returns a loading message.
    Example Usage: print(start.name('Bob'))

  • hello(user_name, choice):
    Greets the user based on a predefined choice.
    Example Usage: print(start.hello('Bob', 'a1'))

Parameters

  • user_name (str):
    The name to set for the model.

  • choice (str):
    The greeting choice (a1, a2, or a3).
    a1: "Hello! How may I help you?"
    a2: "Hello. How may I assist you today?"
    a3: "yo. what's up."
